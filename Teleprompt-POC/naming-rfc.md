# Teleprompter Naming RFC

## Requirements
- Name should suggest smarter teleprompting (timeline/strips/AI cues).
- Domain availability preferred ( .com or .app ).
- Avoid conflicts with existing trademarks (USPTO search).
- Easy to pronounce internationally.

## Candidates to Explore
1. **PromptStrata** – inspired by layered strips.
2. **TeleStrips** – direct nod to strip layout.
3. **PromptGlide** – smooth reading motion.
4. **CueRail** – rail metaphor like WhoWhe timeline.
5. **ScriptOrbit** – orbiting context view.
6. **TelPro-Bro** – playful “teleprompter browser.”
7. **iBroTP** – In-Browser TelePrompter acronym.

## Tasks for AI Assistants
- For each candidate, search USPTO/TESS, domain availability (whois or registrar), and mention any obvious collisions.
- Suggest alternative names if conflicts arise.
- Provide pronunciation tips for top options.
